# 3. Tools & Materials

*   Hex drivers, soldering iron, heat gun (for inserts), flush cutters
*   Glue stick / adhesion promoter for nylon
*   CF cutting tools + PPE
*   Filament dryer or alternative drying method
*   Epoxy/adhesive mixing gear
*   Flat surface + **spacer** for arm bonding
