# 6. Carbon Arm Preparation

![ISO Arm Explode](/img/drone/iso_arm_explode_2.png)
![ISO Arm Explode](/img/drone/Iso_arm_explode.png)
![ISO Arm Reveal](/img/drone/ISO_arm_reveal.png)
![ISO Hidden Line](/img/drone/Iso_hidden_line.png)
![Tumble Annotated](/img/drone/Tumble2.532annotate2.PNG)

* Cut CF tubes to spec (e.g. **133 mm** and **157 mm** — BOM is source of truth)

* Deburr and **wash/wipe with IPA**

* Lightly **sand/scuff** CF bond areas

* Lightly **sand/scuff** inside the printed sockets

* Set up a **flat bonding station** and prepare a **small spacer** for the arm-boss end (because the two printed ends don’t sit at the same height)
