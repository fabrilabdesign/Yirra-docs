# 11. Troubleshooting

* Arm not straight → check bonding stage / spacer use

* Chassis core won’t close → remove leftover internal supports

* GPS not locking → ensure it’s in the nose pocket, not co-routed with XT60

![GPS Reveal](/img/drone/GPS_reveal.png)

* Nylon parts warping → reprint with proper drying + better bed adhesion
